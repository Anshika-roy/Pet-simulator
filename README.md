🐾 Virtual Pet Web App

A fun interactive virtual pet game built using HTML, CSS, and JavaScript.
Choose your pet, interact with it using commands, and keep it happy and alive! 🎮

---

🚀 Features

- 🐶 Choose your pet (Dog, Cat, more coming...)
- 🍖 Feed, 🎾 Play, 😴 Sleep interactions
- 📊 Real-time pet stats (Hunger, Mood, Energy)
- 💾 Save progress using LocalStorage
- ⌨️ Command-based interaction system
- 🎭 Dynamic responses based on actions

Command| Action

sleep| Restores energy
